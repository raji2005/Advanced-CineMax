# 🎬 CineMax - Advanced Movie Booking Dashboard

A professional React admin dashboard for movie booking management.

![React](https://img.shields.io/badge/React-18.2.0-blue?logo=react)
![Tailwind](https://img.shields.io/badge/Tailwind-3.4.1-06b6d4?logo=tailwindcss)
![Vite](https://img.shields.io/badge/Vite-5.0.8-purple?logo=vite)

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📊 Dashboard | Real-time stats, charts, recent bookings |
| 🎬 Movies | Full CRUD, search, filter, pagination |
| 🎫 Bookings | Seat selection, show timings, cancellations |
| 🎭 Shows | Schedule, capacity, pricing tiers |
| 👥 Users | Role-based access, activity logs |
| 💰 Revenue | Sales reports, payment tracking |
| 📈 Analytics | 10+ chart types, trend analysis |
| ⚙️ Settings | System config, notifications |
| 📤 Reports | Export PDF, CSV, Excel |

## 🛠️ Tech Stack

- **React 18** - UI Framework
- **Tailwind CSS** - Styling
- **React Router** - Navigation
- **Recharts** - Data Visualization
- **Lucide Icons** - Icons
- **Vite** - Build Tool

## 🚀 Quick Start

```bash
git clone https://github.com/YOUR_USERNAME/cinemax-dashboard.git
cd cinemax-dashboard
npm install
npm run dev
```

Open `http://localhost:5173`

## 📦 Scripts

```bash
npm run dev       # Development server
npm run build     # Production build
npm run preview   # Preview build
```

## 📁 Structure

src/
├── components/   # Reusable UI components
├── pages/        # 6 complete pages
├── context/      # State management
└── utils/        # Validation 

## ✅ Validation

11 field validations with real-time error messages including title, genre, duration, rating, price, and more.

## 🚀 Deploy

```bash
# Vercel
npm i -g vercel && vercel

# Netlify
npm run build  # drag dist/ to netlify.com
```
